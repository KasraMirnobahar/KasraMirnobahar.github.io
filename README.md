# Kasra Mirnobahar — Academic Portfolio

Personal academic portfolio site, live at **[kasramirnobahar.github.io](https://kasramirnobahar.github.io)**.

## About

I am a Research Assistant at the University of Glasgow (Philiastides Lab, School of Psychology & Neuroscience), working at the intersection of social media, neuroimaging, and computational modelling of behaviour. This site is a single-page portfolio covering my research themes, projects, publications, talks, and CV.

## Built with

Plain HTML, CSS, and JavaScript, no frameworks, no build tools, no dependencies. Deploys directly via GitHub Pages.
**Designed with Claude.ai**
